# Travelogue

旅の記録を残すためのサイト。

- Home: このページ（ピックアップや最近の旅行を表示）
- Map: 訪れた場所や移動ルートを地図上にプロット
- Travels: 各旅行（1旅＝1ページ）を保存
- Museums: 博物館・美術館・動物園・水族館などの記録

## 使い方（簡易）

- 新しい旅行は `docs/Travel/` に新しい Markdown ファイルを作成（テンプレート: `travel-template.md`）。
- 博物館等の記録は `docs/Museum/` に作成（テンプレート: `museum-template.md`）。
- 画像や地図データは `docs/assets/` に整理。

（ここは後でおすすめの旅行カードや最新記事リストを自動化して表示する）
