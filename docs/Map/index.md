# Map

ここでは訪れた場所や移動ルート（歩いた道）を地図上に表示する。

最小実装の流れ：

- 旅行ごとに GeoJSON（または GPX を GeoJSON に変換）を `docs/assets/maps/` に置く
- このページで Leaflet を使って複数の GeoJSON を読み込み、色分け・レイヤ切替をする

（簡易サンプル — 実際に動かすには HTML の許可やカスタムスクリプトの設置が必要）

---

将来的には旅行ごとのレイヤをトグルで切り替えたり、クリックで旅行ページに遷移できるようにする。
